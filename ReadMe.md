# 시나리오
임의의 사용자가 최초에 10000 포인트를 가지고, 1번 요청 시 100 포인트가 차감되도록 총 200회의 요청을 보내려 한다. <br>
결과는 100번의 요청만 허락되어 최종 포인트는 0원이어야 하고, 나머지 100번의 요청은 잔액부족으로 실패해야 한다.<br>
이때, 최대 10번까지 동시 요청이 가능한 경우 결과는 어떻게 될까?
